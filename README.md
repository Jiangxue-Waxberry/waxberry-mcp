# MCP 服务管理平台

## 项目简介
MCP 服务管理平台是一个专为注册和管理各种 MCP 服务而设计的综合平台。它旨在为 MCP 服务的管理、监控和维护提供统一接口，使服务管理更加高效和便捷。

## 开发计划
- [ ] 基础服务注册功能
- [ ] 服务管理界面
- [ ] 基础监控功能
- [ ] 用户权限管理
